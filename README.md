# 9th Island — Mobile-first UI prototype

This branch contains a client-only, mobile-first React + TypeScript + Tailwind prototype using realistic mock data. It's intended as a clickable, production-quality prototype you can show to community members before wiring up a backend like Supabase.

How to run

1. Install dependencies: npm install
2. Run dev server: npm run dev
3. Open http://localhost:5173

What is included

- Mobile-first responsive UI (Home, Vendors, Vendor detail, Gatherings, Events, Map placeholder)
- Client-side mocked auth (localStorage) to simulate sign-in
- Realistic mock data in src/data/mock.ts
- Tailwind CSS for rapid styling
